O que fizemos:
	Usamos a mosma forma de multiplicar a matiz que a otimizada linearmente mas alteramos ela para fazer o uso das instruçoes
    AVX e FMA, a AVX foi usada na parte de multiplicacao por escalar e a FMA para executar a multiplicaçao de matrizes.


Como executar:
	Para nossa maior comodidade, utilizamos o Makefile. Os seguintes comandos foram inseridos no mesmo:

	all:
		clear
		gcc -Wall matrix_lib_test.c headerFiles/matrix_lib.c headerFiles/file_reader.c headerFiles/time.c -o main
		./main 5.0 2048 2048 2048 2048 src/A_2048.dat src/B_2048.dat src/result1.dat src/result2.dat

Esse aquivo "file_reader.c" foi criado para conter as funções para gerenciar os arquivos necessários para obter as matrizes, enquanto
os outros, seguem a especificação proposta.
